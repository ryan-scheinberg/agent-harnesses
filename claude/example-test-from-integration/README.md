# Flask Hello World

A minimal Flask web app with a single `GET /` route that returns `Hello, World!`.

## Install

```bash
pip install -r requirements.txt
```

## Run

```bash
flask run
```

Or alternatively:

```bash
python app.py
```

Then visit [http://localhost:5000/](http://localhost:5000/) or run:

```bash
curl http://localhost:5000/
```

## Test

```bash
pytest
```
